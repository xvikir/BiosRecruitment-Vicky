## 📌 Sections

### 🧩 Python Basics
Foundational Python tasks covering core programming concepts such as loops, conditionals, functions, and data structures. These formed the base for tackling more advanced challenges.

### 🔓 CryptoHack Challenges
Solutions to CryptoHack's introductory cryptography challenges. Topics covered include:
- ASCII & character encoding
- Hexadecimal and Base64 encoding/decoding
- XOR operations and associativity
- Brute-force single-byte XOR
- Repeating-key XOR (known-plaintext attack)

> Full write-ups with explanations and code in [`cryptohack/README.md`](./cryptohack/README.md)

### 🌐 Socket Programming
Hands-on work with Python's `socket` module, covering:
- Setting up TCP client/server connections
- Sending and receiving data over a network
- Practical application of sockets in a security context

> Details and code in [`socket-programming/README.md`](./socket-programming/README.md)
